EN

You are tasked with creating a C program to manage a small collection of books. The program should be able to store details for several books, save them to a file, and then load them back.

First, define a `struct` named `Book` using `typedef`. This structure should contain fields for `title` (a character array capable of holding up to 50 characters, including the null terminator), `author` (a character array for up to 30 characters), and `page_count` (an integer).

Your program should then prompt the user to enter details for 3 books. Store these details in an array of `Book` structs. After collecting the input, write the entire array of books to a binary file named `"library.dat"`. Finally, read the book data back from `"library.dat"` into a *new* array of `Book` structs and display the details of the loaded books to the console. Ensure no pointers or dynamic memory allocation are used in your solution.

TR

Küçük bir kitap koleksiyonunu yönetmek için bir C programı oluşturmanız istenmektedir. Program, birkaç kitabın ayrıntılarını saklayabilmeli, bunları bir dosyaya kaydedebilmeli ve daha sonra geri yükleyebilmelidir.

Öncelikle, `typedef` kullanarak `Book` adında bir `struct` tanımlayın. Bu yapı, `title` (null sonlandırıcı dahil 50 karaktere kadar tutabilen bir karakter dizisi), `author` (30 karaktere kadar bir karakter dizisi) ve `page_count` (bir tam sayı) alanlarını içermelidir.

Programınız daha sonra kullanıcıdan 3 kitabın ayrıntılarını girmesini istemelidir. Bu ayrıntıları bir `Book` struct dizisinde saklayın. Girdileri topladıktan sonra, tüm kitap dizisini `"library.dat"` adlı bir ikili dosyaya yazın. Son olarak, kitap verilerini `"library.dat"` dosyasından *yeni* bir `Book` struct dizisine geri okuyun ve yüklenen kitapların ayrıntılarını konsola yazdırın. Çözümünüzde işaretçi veya dinamik bellek ayırma kullanmadığınızdan emin olun.
