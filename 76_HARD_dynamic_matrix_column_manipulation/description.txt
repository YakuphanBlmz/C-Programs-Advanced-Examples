EN

In C programming, managing dynamic two-dimensional arrays, often referred to as matrices, is a common task. However, when the dimensions of these arrays need to change dynamically after initial allocation, the complexity significantly increases, especially when operations affect the internal structure across all rows. This problem challenges you to implement a dynamic 2D integer matrix where columns can be inserted or deleted, requiring careful memory management and pointer manipulation.

You are tasked with creating a system that simulates a dynamic rectangular matrix. Initially, the matrix will be allocated with a fixed number of rows and columns using `malloc`. Each row, which is itself a dynamic array, will be allocated separately. The core challenge lies in the functions you will implement to modify the matrix's column count. Inserting a column means every single row must be reallocated to a larger size, and existing elements must be shifted to make space for the new column. Similarly, deleting a column requires reallocating every row to a smaller size and shifting elements to fill the gap left by the removed column.

Your solution must include the following functionalities: 1. A function to dynamically allocate an `M x N` integer matrix. 2. A function to print the current state of the matrix. 3. A function `insertColumn(int **matrix, int rows, int *cols, int columnIndex, int defaultValue)` that adds a new column at a specified `columnIndex`, filling new cells with `defaultValue`, and updating the total column count. 4. A function `deleteColumn(int **matrix, int rows, int *cols, int columnIndex)` that removes the column at a specified `columnIndex`, updating the total column count. 5. A function to properly free all dynamically allocated memory. Crucially, you must not use `struct`, `typedef`, or any file I/O operations. All memory operations should be handled using `malloc`, `realloc`, and `free` exclusively, demonstrating a deep understanding of raw pointer arithmetic and dynamic memory management.

TR

C programlamada, genellikle matrisler olarak adlandırılan dinamik iki boyutlu dizileri yönetmek yaygın bir görevdir. Ancak, bu dizilerin boyutları ilk tahsisten sonra dinamik olarak değişmesi gerektiğinde, özellikle işlemler tüm satırlardaki iç yapıyı etkilediğinde karmaşıklık önemli ölçüde artar. Bu problem, işaretçi manipülasyonu ve dikkatli bellek yönetimi gerektiren, sütunların eklenebileceği veya silinebileceği dinamik bir 2B tamsayı matrisi uygulamanızı zorlamaktadır.

Bir dinamik dikdörtgen matrisi simüle eden bir sistem oluşturmanız istenmektedir. Başlangıçta, matris `malloc` kullanılarak belirli sayıda satır ve sütunla tahsis edilecektir. Kendisi dinamik bir dizi olan her satır ayrı ayrı tahsis edilecektir. Temel zorluk, matrisin sütun sayısını değiştirmek için uygulayacağınız işlevlerde yatmaktadır. Bir sütun eklemek, her bir satırın daha büyük bir boyuta yeniden tahsis edilmesi gerektiği ve yeni sütun için yer açmak üzere mevcut öğelerin kaydırılması gerektiği anlamına gelir. Benzer şekilde, bir sütunu silmek, her bir satırı daha küçük bir boyuta yeniden tahsis etmeyi ve kaldırılan sütunun bıraktığı boşluğu doldurmak için öğeleri kaydırmayı gerektirir.

Çözümünüz aşağıdaki işlevleri içermelidir: 1. Bir `M x N` tamsayı matrisini dinamik olarak tahsis eden bir işlev. 2. Matrisin mevcut durumunu yazdıran bir işlev. 3. Belirtilen `columnIndex` konumuna yeni bir sütun ekleyen, yeni hücreleri `defaultValue` ile dolduran ve toplam sütun sayısını güncelleyen `insertColumn(int **matrix, int rows, int *cols, int columnIndex, int defaultValue)` işlevi. 4. Belirtilen `columnIndex` konumundaki sütunu kaldıran ve toplam sütun sayısını güncelleyen `deleteColumn(int **matrix, int rows, int *cols, int columnIndex)` işlevi. 5. Dinamik olarak tahsis edilmiş tüm belleği düzgün bir şekilde serbest bırakan bir işlev. Kesinlikle, `struct`, `typedef` veya herhangi bir dosya G/Ç işlemi kullanmamalısınız. Tüm bellek işlemleri yalnızca `malloc`, `realloc` ve `free` kullanılarak ele alınmalı, ham işaretçi aritmetiği ve dinamik bellek yönetimi konusundaki derin anlayışınızı göstermelisiniz.
