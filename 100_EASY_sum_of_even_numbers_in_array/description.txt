EN

Write a C program that reads 5 integer values into an array. Calculate and print the sum of all even numbers present in the array. The program should only output the final sum.

TR

C programlama dilinde, 5 adet tam sayı değerini bir diziye okuyan bir program yazın. Dizideki tüm çift sayıların toplamını hesaplayıp ekrana yazdırın. Program yalnızca nihai toplamı çıktı olarak vermelidir.
