EN

Given an array of integers, write a C program to count how many positive numbers are present in the array. A positive number is defined as any number greater than zero. The program should print the total count of positive numbers.

TR

Verilen bir tam sayı dizisinde, dizide kaç tane pozitif sayı olduğunu sayan bir C programı yazın. Pozitif bir sayı, sıfırdan büyük olan herhangi bir sayı olarak tanımlanır. Program, pozitif sayıların toplam sayısını ekrana yazdırmalıdır.
