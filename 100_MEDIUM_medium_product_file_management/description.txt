EN

This problem requires you to implement a simple product management system using C structs and binary file I/O. You will define a `Product` structure using `typedef`, which includes fields for a product's ID, name, price, and stock quantity.

The program should first prompt the user to enter details for a specified number of products. As each product's information is entered, it should be immediately written to a binary file named `products.dat`. After all products have been saved, the program must then read all the product records back from the `products.dat` file. Finally, it should display the details of each retrieved product to the console. Ensure that no pointers or dynamic memory allocation are used at any point in your solution.

TR

Bu problem, C dilinde `struct` yapıları ve ikili dosya giriş/çıkış (file I/O) kullanarak basit bir ürün yönetim sistemi uygulamanızı gerektirir. Bir ürünün kimliği (ID), adı, fiyatı ve stok miktarı için alanları içeren bir `Product` yapısını `typedef` kullanarak tanımlayacaksınız.

Program ilk olarak kullanıcıdan belirli sayıda ürün için detayları girmesini istemelidir. Her ürünün bilgisi girildikçe, bu bilgiler hemen `products.dat` adlı bir ikili dosyaya yazılmalıdır. Tüm ürünler kaydedildikten sonra, program `products.dat` dosyasından tüm ürün kayıtlarını geri okumalıdır. Son olarak, geri alınan her ürünün detaylarını konsola yazdırmalıdır. Çözümünüzde hiçbir işaretçi (pointer) veya dinamik bellek ayırma (dynamic memory allocation) kullanılmadığından emin olun.
