EN

Write a C program that initializes an integer array with a predefined set of values. The program should then calculate the sum of all elements present in this array. Finally, print the computed sum to the console.

TR

Önceden tanımlanmış değerlerle bir tam sayı dizisini başlatan bir C programı yazın. Program, bu dizideki tüm elemanların toplamını hesaplamalıdır. Son olarak, hesaplanan toplamı konsola yazdırmalıdır.
