EN

Write a C program that calculates the sum of all elements in a given integer array. The program should initialize an array with a fixed set of integer values and then compute and print their total sum.

TR

Verilen bir tam sayı dizisindeki tüm elemanların toplamını hesaplayan bir C programı yazın. Program, sabit bir dizi tam sayı değeri ile bir diziyi başlatmalı ve ardından toplamlarını hesaplayıp ekrana yazdırmalıdır.
